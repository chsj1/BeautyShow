## 应用简介
美女神社本应用集合了各行各业各式各样的美女、女神，中国内地、港澳台、亚洲日韩、欧美，演艺明星、歌手、模特等等可供宅男们在平时休闲的时候放松欣赏一下美丽的女神图集。科学研究表明，多看美女有助于延年益寿。
